# Blinkit Homepage UI Clone

A high-fidelity, fully responsive Blinkit.com homepage recreation built with React.js, Tailwind CSS, and Framer Motion.

## 🚀 Features

- **Pixel-Perfect UI**: Matches Blinkit's modern, clean, and vibrant design.
- **Responsive Layout**: Seamlessly works on Desktop, Tablet, and Mobile devices.
- **Live Search**: Real-time product filtering as you type.
- **Cart Management**: State-managed cart with dynamic badge updates and toast notifications.
- **Interactive Carousel**: Professional promotional banners with smooth autoplay.
- **Category Navigation**: Horizontal scrollable category list with hover effects.
- **Smooth Animations**: Powered by Framer Motion for a premium feel.
- **Skeleton Loaders**: Polished loading states to simulate real-world data fetching.
- **Performance Optimized**: Lazy loading for images and modular component architecture.

## 🛠️ Tech Stack

- **Frontend**: React.js (Hooks + Context API)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: React Icons / Lucide React
- **Notifications**: React Hot Toast
- **Build Tool**: Vite

## 📦 Folder Structure

```text
src/
├── components/      # Reusable UI components
├── context/         # Cart and Global state management
├── data/            # Mock JSON data (realistic grocery items)
├── pages/           # Page layouts
├── styles/          # Global styles
├── App.jsx          # Root component
└── main.jsx         # Entry point
```

## 🏗️ Getting Started

1. **Clone the project**
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm run dev
   ```
4. **Build for production**:
   ```bash
   npm run build
   ```
